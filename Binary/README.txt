Kebutuhan Sistem
Minimal OS Android 4.0 Ice Cream Sandwich


Panduan Instalasi
1. Download file Yasiin.apk yang tersedia Repositories “Yasiin/Yasiin-dan-Tahlil”  di github. Yakni pada folder Source code -> Binary -> Yasiin.apk. File dapat di download melalui device Android maupun PC. Apabila didownload melalui PC harus mengirimkan File Instalasi pada device Android melalui bluetooth ataupun kabel data.
2. Buka file "Yasiin.apk" pada device Android.
3. Klik button "Instal".
4. Proses Instalasi.
5. Pilih "Buka" apabila telah selesai menginstal.
6. Apabila penginstalan selesai, akan ditampilkan menu home aplikasi
